# RentFlow - Микросервисная платформа аренды инструментов

## Описание проекта

RentFlow - это платформа для аренды инструментов, построенная на микросервисной архитектуре. Проект разделен на несколько независимых сервисов, каждый из которых отвечает за определенную функциональность.

## Архитектура проекта

Проект состоит из следующих модулей:

- **common** - общий модуль с общими классами и утилитами
- **user-service** - сервис для управления пользователями
- **tool-service** - сервис для управления инструментами

## Технологический стек

- Java 21
- Spring Boot 3.5.11
- Spring Data JPA
- PostgreSQL
- Liquibase
- Lombok
- Maven
